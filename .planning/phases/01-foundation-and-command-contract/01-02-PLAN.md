## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| CLI process ↔ config file | Token and default repo are persisted locally | secrets, repo names, user settings |
| CLI process ↔ environment | Process environment can override config values | token, repo, output mode |

## Threat Register

| Threat ID | Category | Component | Disposition | Mitigation | Status |
|-----------|----------|-----------|-------------|------------|--------|
| T-1-03 | Information Disclosure | local config storage | mitigate | enforce restrictive file permissions and redact token values in display paths | open |
| T-1-04 | Tampering | config precedence | mitigate | centralize file/env resolution and cover precedence rules with tests | open |
</threat_model>
